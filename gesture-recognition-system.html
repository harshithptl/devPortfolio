Gesture Recognition System

A system that leverages real-time hand gesture detection to automate VLC media player.

Code Link

Project Overview

This system employs a custom VGG-16-based convolutional neural network model, trained on data manually collected from my own hand gestures. OpenCV is utilized for robust image processing to accurately detect and segment gesture data from a live video feed.

The recognized gestures then trigger specific automation commands for VLC Media Player—such as pause/play, mute, and volume adjustments—forming the foundation of a seamless, gesture-controlled automation system.
